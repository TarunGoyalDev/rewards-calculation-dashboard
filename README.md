📊 **Rewards Calculation Dashboard**

A modern React-based dashboard application to analyze customer transactions and reward points.
The application provides insights into customer behavior, transactions, and reward distribution with filtering, pagination, and analytics.

⚙️ **Setup Instructions**

**Install Dependencies**

npm install

**Run the App**

npm start

**Run Tests**

npm test

**Build For Production**

npm run build


🚀 **Tech Stack**

- React.js
- JavaScript (ES6+)
- CSS
- Jest + React Testing Library

🚀 **Features**


📈 Dashboard

Provides a high-level overview of customer activity and reward metrics through key performance indicators such as total customers, transactions, and points. It includes month-based filtering to analyze trends over time. The dashboard also highlights top-performing customers, cities, and months based on transaction amounts. Designed to deliver quick, actionable insights in a clean and intuitive layout.




👥 Customers

Displays a detailed list of customers in a structured table with attributes like name, age, gender, city, and total reward points. Supports advanced filtering including search, demographic filters, and points range to quickly find specific customer segments. Pagination ensures efficient handling of large datasets. The interface is optimized for clarity and ease of use.





💳 Transactions

Presents transaction data in a tabular format including date, customer name, amount, and reward points earned. Users can filter transactions using date range, amount range, and points criteria for targeted analysis. Pagination enables smooth navigation through large volumes of data. The design focuses on readability and quick data interpretation.
